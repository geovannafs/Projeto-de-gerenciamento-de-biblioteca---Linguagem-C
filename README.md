Gerenciamento de biblioteca 🚀

Software para gestão bibliotecária que automatiza o controle de exemplares e o monitoramento de prazos de empréstimos utilizando a linguagem C.

📌 Sobre o Projeto

Esse projeto foi criado para resolucionar uma atividade da faculdade na matéria de Linguagem e Técnicas de Programação.

🛠️ Tecnologias Utilizadas
- Linguagem C
- Biblioteca tal
- Compilador GCC

⚙️ Funcionalidades
- Persistência de Dados: Salva e carrega automaticamente informações de livros, usuários e empréstimos em arquivos de texto (.txt).
- Gestão de Datas: Utiliza a biblioteca time.h para gerar datas de cadastro e calcular prazos de devolução (7 dias).
- Controle de Status: Sistema inteligente que altera o estado do livro (Disponível/Emprestado) e do empréstimo (Ativo/Atrasado/Devolvido).
- Pesquisa Avançada: Busca de livros e usuários por múltiplos critérios (código, título, nome, matrícula) com suporte a case-insensitivity.
- Relatórios Acadêmicos: Geração de estatísticas de livros mais emprestados e listagem de usuários inadimplentes.
- Robustez: Tratamento de buffer de entrada (LimparBuffer) e validações de dados para evitar erros de execução.
